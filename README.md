# wgr-cast

Public hosting for the **Weird Good Radio** Google Cast *Styled Media Receiver* skin.

This repo exists only so the skin can be served over HTTPS via GitHub Pages
(the main app repo is private, and GitHub Pages requires a public repo on the
free plan). It contains brand assets and CSS only — no application code.

Served at:

- Skin (paste this as the **Skin URL** in the Cast Developer Console):
  `https://adam-today.github.io/wgr-cast/receiver.css`
- Fonts: `https://adam-today.github.io/wgr-cast/fonts/SofiaSansSemiCondensed-*.ttf`
- Logo: `https://adam-today.github.io/wgr-cast/logo.png`
- Splash: `https://adam-today.github.io/wgr-cast/splash.png`

`receiver.css` is the canonical copy of the file kept in the app repo at
`server/public/cast/receiver.css`; keep the two in sync.

Sofia Sans Semi Condensed is licensed under the SIL Open Font License.
